# gulp-webpack-example

利用gulp实现本地服务、mock数据、css预处理、js模块化、打包压缩、版本号替换等功能.

ps：
- 构建多页面前端项目
- CSS预编译采用SASS
- JS基于commonjs规范模块化编译
- MOCK用fs、url进行配置


目前用到：
- gulp
- gulp-clean
- gulp-minify-css
- gulp-rev
- gulp-rev-collector
- gulp-sass
- gulp-uglify
- gulp-webpack
- gulp-webserver
- imports-loader
- node-sass
- vinyl-named
